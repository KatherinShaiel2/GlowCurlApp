# Glow Curl App

## Descripción

Glow Curl App es una aplicación web desarrollada como proyecto formativo del programa Análisis y Desarrollo de Software (ADSO) del SENA.

La aplicación está diseñada para apoyar la administración de un salón especializado en el cuidado del cabello rizado, permitiendo gestionar clientes, productos, servicios, citas y diferentes procesos administrativos mediante una interfaz intuitiva.

---

## Objetivo

Desarrollar una aplicación web que facilite la gestión de la información de clientes y de la empresa, mejorando la organización de los servicios y la experiencia de los usuarios.
